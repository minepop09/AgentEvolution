# AgentEvolution - Agent 问题处理操作系统

> 让 Agent 具备可移植、可进化的问题处理能力

---

## 项目概述

构建一个**问题处理操作系统**，可以加载到任何 Agent 的 Soul 里，使其具备：
- 深度问题分析能力
- 专业领域知识学习能力
- 工具审核与最优解选择能力
- 自主进化能力（Skill编排 + 工具权重）

## 核心架构

```
Agent Soul
├── 问题处理OS（7个模块流水线）
│   ├── 问题预处理 → 问题认知 → 问题分析
│   ├── 工具审核（SlowMist审计）→ 工具安装
│   └── Todo计划（贯穿全程）
├── 知识库（纯文件方案，Markdown）
└── 技能包（按领域加载）
```

---

## 七大模块

1. **问题预处理模块** - 拆解问题，判断类型
2. **问题认知模块** - 三层知识探测（置信度自评 → 知识库验证 → 主动探测）
3. **知识库存储模块** - 领域知识存取
4. **问题分析模块** - 深度分析 + 方法论选择
5. **工具审核模块** - 评分前2 + SlowMist审计 + Low拒绝
6. **工具安装模块** - 安装通过审核的工具
7. **Todo计划模块** - 贯穿全程的任务状态管理
