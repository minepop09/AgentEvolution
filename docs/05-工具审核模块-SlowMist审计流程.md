# 工具审核模块：SlowMist 安全审计流程

> 日期：2026-05-19

## 核心流程

```
枚举候选工具
    ↓
获取网络综合评分（取前2）
    ↓
┌─────────────────────┐
│  SlowMist 安全审计    │
└────────┬────────────┘
         ↓
    ┌────┴────┐
    ↓         ↓
  通过       Low
   ↓          ↓
  安装     不安装，切换
            到下一个候选
```

---

## 评分规则

| 评分 | 含义 | 动作 |
|------|------|------|
| High | 综合评分最高 | 进入审计 |
| Medium | 综合评分次之 | 进入审计 |
| Low | 评分低 | 不安装，告知用户 |

---

## 安装策略

- **最多安装2个备选工具**（按评分排序）
- 如果第一个安装成功 → 完成
- 如果第一个安装失败 → 尝试第二个
- 如果第二个也失败 → 告知用户"工具安装全部失败，请手动处理"

---

## 输出格式

```
工具审核报告：
- 候选工具：Tool A (评分9.2), Tool B (评分8.7)
- SlowMist审计：Tool A → 通过，Tool B → Low（不安装）
- 安装结果：Tool A → 成功
```

---

## 与 SlowMist Agent Security skill 的集成

审计时调用 `slowmist-agent-security` skill，对候选工具进行：
- 代码安全扫描
- 依赖漏洞检测
- 权限过度申请检测
- 恶意行为模式识别

审计结果评分：High / Medium / Low
