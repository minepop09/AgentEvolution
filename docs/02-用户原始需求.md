# 用户原始需求

> 日期：2026-05-19

## 核心思路

AI学习处理问题的思路，分两步：

1. **深度分析阶段**：AI先深度分析问题是什么、怎么处理（方法论），找解决问题的好的方法论
2. **工具阶段**：根据深度分析结果，找处理问题的专业性工具，包括：
   - 在哪找工具
   - 可能多个子问题需要不同工具
   - 如何判断找到的工具是最优解
